# Growthik Media Brand Asset Library

This folder is the organized source library for all Growthik Media logo assets.

## Brand Colors
- Primary Red: `#ed1c24`
- Primary Black: `#231f20`
- White: `#ffffff`

## Folder Structure
- `logos/`: main logo files for website, print prep, and transparent usage
- `favicons/`: browser, mobile, and shortcut icons
- `manifest/`: PWA manifest and install icons
- `palette/`: reusable color reference files

## File Naming Logic
- `growthik-media-logo-transparent-primary.png`
  - Main transparent logo for website use
  - Uses red `#ed1c24`, black `#231f20`, and transparent background
- `growthik-media-wordmark-primary.svg`
  - Vector wordmark source for crisp scaling
- `growthik-media-brand-mark-primary.svg`
  - Vector brand mark only
- `growthik-media-brand-mark-primary.png`
  - Raster brand mark for flexible UI use
- `growthik-media-favicon.*`
  - Browser favicon set
- `growthik-media-apple-touch-icon-180x180.png`
  - iOS home screen icon
- `growthik-media-site.webmanifest`
  - PWA manifest source
- `growthik-media-pwa-icon-192x192.png`
  - Android / install icon
- `growthik-media-pwa-icon-512x512.png`
  - Large install icon

## Live Website Assets
The public website serves branding assets from `public/brand/`.
This `app/assets/images/brand/` folder is the internal organized source library.
